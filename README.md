## Installation

**Via [NPM](https://www.npmjs.com):**

[![NPM](https://nodei.co/npm/symvl.png?downloads=true&downloadRank=true&stars=true)](https://www.npmjs.com/package/symvl)

```
npm install symvl
```

**Via [Yarn](https://yarnpkg.com):**

```
yarn add symvl
```
